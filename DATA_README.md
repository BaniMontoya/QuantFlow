# Instrucciones de Datos (Data Instructions)

El archivo de entrenamiento original era demasiado grande para GitHub, así que se subió en partes.

### Para reconstruir el archivo (Linux / Mac / Git Bash):
```bash
cat QuantFlow_TrainingData_part_* > QuantFlow_TrainingData.csv
```

### Para reconstruir el archivo (Windows CMD):
```cmd
copy /b QuantFlow_TrainingData_part_* QuantFlow_TrainingData.csv
```
